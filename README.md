# Outreachy 32 - Wikimedia Foundation
Outreachy contribution period tasks - March, 2026

## Task 1 - JavaScript

In the problem set, an HTML file a list of articles, each with a `page_id`, `creation_date` and `title`. The task is to update the html file to correctly display the articles in a clean readable format: `Article "ARTICLE TITLE" (Page ID PAGEID) was created at MONTH DAY, YEAR.`.

## Task 2 - Python

This task is about getting the status code of a list of URLs from a .csv file and printing the status code and the url to the console in the format `(STATUS CODE) URL`.
